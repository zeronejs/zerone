# vue3-template

1. pnpm create zerone

# 项目技术栈

1. 打包：vite
2. 语言：ts
3. 框架：vue3
4. css：tailwindcss
5. ui框架：element-plus / giime
6. 状态管理：pinia
7. 包管理：pnpm
8. node 18.12以上

## 项目安装

```sh
pnpm install
```

### 本地运行

```sh
pnpm dev
```

### 正式打包

```sh
pnpm build
```

### 测试打包

```sh
pnpm build:test
```
